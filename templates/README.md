# Poster Templates

Ready-to-use poster templates for the Holistic AI x UCL Hackathon 2025.

## Available Templates

- **`poster_template_A3.pdf`** - PDF template (A3 size) - Ready to use or edit in Adobe Acrobat/Illustrator
- **`poster_template_A3.pptx`** - PowerPoint template (A3 size)

All templates include:

- Official hackathon branding and header
- Background design
- Blank content area for your project

## Requirements

- **Format**: Single page PDF
- **Size**: A3 (recommended) or A4 (acceptable)
- **Content**: Your project content

## How to Use

### Option 1: PDF Template (Recommended)

1. **Open `poster_template_A3.pdf`** in Adobe Acrobat, Illustrator, or any PDF editor
2. **Add your content** directly to the PDF
3. **Save** - Already in PDF format, ready to submit!

### Option 2: PowerPoint Template

1. **Open `poster_template_A3.pptx`** in PowerPoint
2. **Add your content**:
   - Project title and team name
   - Problem statement and solution
   - Key features and results
   - Visuals (diagrams, screenshots, charts)
   - Technical details
3. **Export as PDF**:
   - PowerPoint: File → Export → Create PDF
   - **Recommended**: 300 DPI for high quality

## Submission

Export as high-quality PDF (300 DPI recommended) and submit via Devpost.

## Tips

- Keep text concise and readable
- Use visuals to illustrate your solution
- Highlight key achievements and metrics
- Ensure all team members are credited
- Proofread before submission
