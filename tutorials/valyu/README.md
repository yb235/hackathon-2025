## ğŸ’¡ Key Concepts

### Tools vs Retrievers

| Feature | Tools | Retrievers |
|---------|-------|------------|
| **Use Case** | AI agents | RAG applications |
| **Returns** | Dict with results | List of Documents |
| **Integration** | Agent tool calling | RAG chains |
| **Examples** | ValyuSearchTool | ValyuRetriever |

### Search vs Contents

| Feature | Search | Contents |
|---------|--------|----------|
| **Input** | Query string | List of URLs |
| **Purpose** | Find information | Extract from known URLs |
| **Discovery** | âœ… Yes | âŒ No |
| **When to Use** | Research, discovery | Known sources |

## ğŸ› ï¸ Quick Reference

### Basic Search

```python
from langchain_valyu import ValyuSearchTool

tool = ValyuSearchTool()
result = tool._run(query="quantum computing", max_num_results=5)
```

### Content Extraction

```python
from langchain_valyu import ValyuContentsTool

tool = ValyuContentsTool(summary=True)
result = tool._run(urls=["https://example.com/article"])
```

### RAG Retriever

```python
from langchain_valyu import ValyuRetriever
from langchain.chains import RetrievalQA
from langchain_openai import ChatOpenAI

retriever = ValyuRetriever(k=5)
llm = ChatOpenAI(model="gpt-4")
qa = RetrievalQA.from_chain_type(llm=llm, retriever=retriever)
```

### Complete Agent

```python
import sys
sys.path.insert(0, '../core')

from react_agent import create_react_agent
from langchain_valyu import ValyuSearchTool

agent = create_react_agent(
    tools=[ValyuSearchTool()],
    model_name='claude-3-5-sonnet'
)

result = agent.invoke({
    'messages': [('user', 'What is quantum computing?')]
})
```
